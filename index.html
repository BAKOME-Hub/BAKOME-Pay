📚 Documentation

Getting Started

1. Install Phantom Wallet from phantom.app
2. Click "Connect Phantom" button
3. Approve the connection in your wallet
4. Start sending payments instantly!

How to Send SOL

1. Connect your wallet
2. Enter recipient's Solana address
3. Enter amount in SOL
4. Click "Send SOL"
5. Approve transaction in Phantom
6. Wait for confirmation (~1 second)

Supported Wallets

✅ Phantom
✅ Solflare
✅ Backpack
✅ Any Solana Standard Wallet

FAQ

Q: Is my data safe?
A: Yes! We never store your private keys or personal data.

Q: What are the fees?
A: Only Solana network fees (~0.00001 SOL).

Q: Can I use on mobile?
A: Yes! Install as PWA or use any mobile wallet.
